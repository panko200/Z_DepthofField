Z軸被写界深度
製作者：Panko200
配布場所：https://github.com/panko200/Z_DepthofField

## 概要

カメラから見た距離と、物体のX,Y,Zの値を比較してピント距離と合致していない場合、ぼかしがかかります。
画像からの推定でぼかしをかけるなどの高度なのはつけてません。重くなると思う。それやるならDepth EstimateとToneCurveかColorRamp、そしてぼかしマップ使ってね

## 使用方法

映像エフェクト「Z軸被写界深度」が追加されます。  
カメラとアイテム座標とのピントの合う距離、「ピント距離」を動かして調整してください。
グループ制御でカメラの後に本プラグインをかけると簡単に扱えます。

## アンインストール方法

1. YMM4 を起動して`ヘルプ(H)`>`その他`>`プラグインフォルダを開く`をクリックする。
2. YMM4 を終了する。
3. `Z_DepthofFieldEffect`という名前のフォルダを削除する。

## 注意点

ゆっくりムービーメーカーv4.48.1.0以上を使用してください。

アイテムの座標によって見え方を変化させるので、アイテムのピクセルごとの座標とそのサイズによる見え方の差異によってのぼかしは実装していません。
